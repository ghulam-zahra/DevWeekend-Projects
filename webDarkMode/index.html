🌙 Dark mode converter

Toggle any page into dark mode · separate files

📄 HTML

This page is built with semantic HTML. The dark mode toggle is powered by a single JavaScript function that toggles a class on the <body>.

🎨 CSS

All styles are in style.css. Dark mode variables are defined inside the .dark-mode class – everything updates smoothly.

⚡ JS

The logic lives in script.js. It listens for the toggle button, applies/removes the dark-mode class, and saves your preference in localStorage.
